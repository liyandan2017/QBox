## 应用预览 | UI

> 图片依次是（多动图，加载较慢，图片不显示刷新页面）：  
> 1.新闻模块(**如果你下载的 apk 没有显示新闻模块，开启方式：设置页长按头部直至出现提醒 > 设置页长按底部直至出现提醒 > 退出重启应用**)  
> 2.文章模块 3.日历模块  
> 4.天气模块 5.颜色画板 6.盒子布局    
> 7.魔力数字排序列表 8.下拉果冻弹跳效果 9.引导页  
> 10.手势图片 11.能力图表格 12.视差图片
